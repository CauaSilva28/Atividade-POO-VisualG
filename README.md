# Atividade-POO-VisualG
Programa com as atividades passadas em aula no VisualG
## Combustível
Nós criamos um carácter chamado comb, para chamar as variáveis "A" de álcool e "G" de gasolina.
Basicamente utilizamos a função "se" para saber se a variável seria "A" ou "G" e dependo do que fosse analisado o programa iria usar os dados de cada variável para completar o que foi pedido.
