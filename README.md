Programa com as atividades passadas em aula no VisualG
# Combustivel
Nós criamos um caracter chamado comb, para chamar as variaveis "A" de álcool e "G" de gasolina.
Basicamente utilizamos a função "se" para saber se a variavel seria "A" ou "G" e dependo do que fosse analisado o programa iria usar os dados de cada variavel para completar o que foi pedido.
